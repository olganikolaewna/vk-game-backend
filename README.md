# VK Game Platform API

Бэкенд для платформы мини-игр с ИИ

## Эндпоинты

- `GET /` - корень API
- `GET /health` - проверка здоровья
- `POST /api/v1/games/sudoku/new` - создать новую игру
- и другие (см. `/docs`)
